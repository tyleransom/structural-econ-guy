\begin{frame}

There is a trade-off between bias and variance

\bigskip{}

\begin{align*}
\underbrace{\mathbb{E}\left(y-\hat{f}(x)\right)^2}_{\text{Expected }MSE\text{ in test set}} &= \underbrace{\mathbb{V}\left(\hat{f}(x)\right)}_{\text{Variance}} + \underbrace{\left[\text{Bias}\left(\hat{f}(x)\right)\right]^2}_{\text{Squared Bias}} + \underbrace{\mathbb{V}\left(\epsilon\right)}_{\text{Irreducible Error}}
\end{align*}

\bigskip{}

\begin{itemize}
\itemsep1.5em
\item<2-> A model with high \textcolor{navy}{bias} is a poor approximation of reality
\item<3-> A model with high \textcolor{navy}{variance} is one that does not generalize well to a new data set
\item<4-> A model is \textcolor{navy}{overfit} if it has low bias and high variance
\item<5-> A model is \textcolor{navy}{underfit} if it has high bias and low variance
\end{itemize}

\end{frame}

\begin{frame}

When the target variable is continuous, we use $MSE$ to measure fit

\bigskip{}

\onslide<2->{
When it is categorical, we use \textcolor{navy}{accuracy} or similar measures
}

\bigskip{}

\begin{itemize}
\itemsep1.5em
\item<3-> Or some combination of specificity and sensitivity
\item<4-> Goal is to not have a good ``fit'' by randomly guessing
\item<5-> So each potential metric penalizes random guessing
\end{itemize}

\end{frame}
